# CodeAlpha_DataVisualization

## 📌 Project Overview
This project focuses on visualizing the Amazon Best Selling Books dataset using different charts and graphs to better understand trends and patterns.

## 📂 Dataset
Amazon Best Selling Books Dataset

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## 📊 Visualizations Included
- Histogram
- Bar Chart
- Countplot
- Scatter Plot
- Pie Chart
- Heatmap
- Line chart

## 🎯 Objective
To transform raw dataset information into meaningful visual insights using data visualization techniques.

## 📌 Conclusion
Data visualization helped identify popular categories, rating distributions, price patterns and relationships between different variables in the dataset.
